# Weather Files

Thư mục này chứa các file weather (.epw) cho EnergyPlus simulation.

## Download Weather File

1. Truy cập: https://energyplus.net/weather
2. Tìm kiếm: **Vietnam, Ho Chi Minh City**
3. Download file: `VNM_Ho.Chi.Minh.488500_IWEC.epw`
4. Lưu vào thư mục này

## Weather Files khác

Bạn có thể download weather files cho các địa điểm khác:
- Hà Nội
- Đà Nẵng
- Các thành phố khác

Nhớ cập nhật tên file trong scenario JSON config.
